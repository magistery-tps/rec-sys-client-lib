Welcome to rec-sys-client-lib's documentation!
==============================================

This library give support for:

* Access to `rec-sys <https://github.com/magistery-tps/rec-sys>`_ via database repositories.
* Access to `rec-sys <https://github.com/magistery-tps/rec-sys>`_ REST API to config and update recommenders data.
* Jobs: Used to build and config similarity matrix (user-user / item-item) required by `rec-sys <https://github.com/magistery-tps/rec-sys>`_ recommenders.
